## Key Patterns Extracted

### Starter Position Strategy
1. First entry = small (2,000 shares)
2. If it goes against you, loss is tolerable
3. If it works, add on confirmation
4. Only size up AFTER proof of strength

### "Pulling Away" Pattern
1. Price consolidates near level
2. Breaks through with volume
3. IMMEDIATELY moves 50+ cents
4. This is the ADD signal

### News Headline Analysis
- AXIL: "Walmart" sounded great but...
- Deeper dig: In-ear hearing plugs = low margin
- Retail product, not tech/crypto = less upside
- Don't get sucked in by headline, read details

### Add Back Rules
1. Only add when stock is obvious
2. Must meet 5 pillars still
3. Starter first, then scale
4. Accept going red briefly on small size
5. Never full size on first entry
