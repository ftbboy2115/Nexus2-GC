# Warrior Trading Transcript Catalogue

This directory contains catalogued transcripts from Ross Cameron's Warrior Trading videos, organized for reference when developing and tuning the Warrior scanner logic.

## Purpose
- Document real trading patterns, entries, and exits
- Capture theme recognition and catalyst evaluation
- Record disqualifier patterns (what to avoid)
- Build training data for scanner improvements

## File Format
Each transcript is stored as a markdown file with:
1. **Metadata** - Date, video title, URL, stock(s) discussed
2. **Trade Summary** - Quick reference table
3. **Entry/Exit Analysis** - Detailed breakdown
4. **Key Patterns** - Extracted rules and observations
5. **Full Transcript** - Complete text for reference
