## Borrow Status Workflow (DETAILED)

### Steps Ross Takes [DIRECT QUOTE sequence]
```
1. Type ticker into broker
2. Check borrow status letter:
   - T (Threshold) = Cannot short at all
   - L (Locate) = Can borrow, check price
   - E (Easy) = Usually avoid (shorts aggressive)

3. If L, check borrow cost:
   - Click "borrow" button
   - Check price per 100 shares
   - Calculate for position size
   - Example: 2.5¢/share × 10k = $150

4. Check if on Short Sale Restriction (SSR)
```
