## Cushion Building Framework

### Daily Cushion Strategy
1. First trade = smallest size (riskiest because no cushion)
2. Build $5k-$10k cushion with small wins
3. THEN take larger size on trades 3, 4, 5, etc.
4. If no cushion builds → stay throttled back

### Weekly Cushion Strategy
Same pattern at weekly level:
- Monday strong → more aggressive Wed/Thu/Fri
- Monday weak → conservative rest of week
- Don't want to risk going red on week with only Friday left
