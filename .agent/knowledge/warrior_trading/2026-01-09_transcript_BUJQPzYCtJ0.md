## Key Patterns Extracted

### Entry Rules
1. Don't hesitate on first entry with good spreads
2. 1-minute pullback after blue sky break is valid entry
3. **Size limit**: Account size limits position (10k shares max on $14 stock)

### Add Rules
1. Don't double down on wildcard setups
2. If you add at break of key level, be ready to cut fast

### Exit Rules
1. **Cut loss immediately** - don't hope for break-even
2. When back to break-even after being up, SELL
3. "Get out earlier than you feel like you should"

### Avoid Rules
1. **Higher priced stocks** ($50+) = much larger losses per share
2. **Chinese stocks with no news** = extra risky
3. **Wildcard trades** = only if you can afford the loss
4. **Friday afternoon** = walk away, try again Monday
